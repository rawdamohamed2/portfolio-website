# Portfolio Website

A personal portfolio website built with React and Bootstrap.  
It showcases projects, skills, and contact information.

## Features

- Responsive design  
- Projects section with live links  
- Skills overview  
- About section  
- Education section  
- Contact section  
- Smooth navigation  
- Light/dark mode toggle  

## Technologies Used

- React  
- Bootstrap  
- HTML  
- CSS  
- JavaScript  
- GitHub
- Git
